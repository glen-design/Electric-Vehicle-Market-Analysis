# Electric Vehicle Market Analysis

## Project Overview
This project analyzes Electric Vehicle (EV) market trends using Power BI. The dashboard provides insights into EV adoption growth, leading manufacturers, popular EV models, and EV type distribution.

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Key Insights
- EV adoption has increased rapidly in recent years
- Tesla leads EV manufacturers in total registrations
- Battery Electric Vehicles dominate the EV market

## Files in this Repository
electric_vehicle_dataset.csv – Dataset used for analysis  
ev_market_dashboard.pbix – Power BI dashboard
